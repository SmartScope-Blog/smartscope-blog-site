    ## рҹ—ӮпёҸ гғЎгӮӨгғігӮ«гғҶгӮҙгғӘж§ӢжҲҗ 
    рҹ“ҡ еӯҰзҝ’гғ»жғ…е ұ
в”ңв”Җв”Җ рҹ“° гғӢгғҘгғјгӮ№гғ»гғҲгғ¬гғігғү
в”ңв”Җв”Җ рҹҢҚ иӢұиӘһеӯҰзҝ’
в””в”Җв”Җ рҹ”— гғӘгғігӮҜйӣҶ

рҹ–ҘпёҸ гӮӨгғігғ•гғ©гғ»йҒӢз”Ё
в”ңв”Җв”Җ гӮ·гӮ№гғҶгғ еҹәзӨҺ (OSзўәиӘҚгҖҒDNSгҖҒSELinux)
в”ңв”Җв”Җ гғ•гӮЎгӮӨгӮўгӮҰгӮ©гғјгғ«гғ»гӮ»гӮӯгғҘгғӘгғҶгӮЈ
в”ңв”Җв”Җ иҮӘеӢ•еҢ–гғ»гӮ№гӮұгӮёгғҘгғјгғӘгғігӮ° (crontab)
в”ңв”Җв”Җ гғҮгӮЈгӮ№гӮҜгғ»гӮ№гғҲгғ¬гғјгӮё
в”ңв”Җв”Җ гғ‘гғғгӮұгғјгӮёз®ЎзҗҶ
в””в”Җв”Җ гӮігғһгғігғүгғ»гғ„гғјгғ«

рҹӨ– AIй–Ӣзҷәгғ»иҮӘеӢ•еҢ–
в”ңв”Җв”Җ Claude Codeз·ҸеҗҲгӮ¬гӮӨгғү (50+ иЁҳдәӢ)
в”ңв”Җв”Җ рҹҺҜ AIгӮЁгғјгӮёгӮ§гғігғҲй–Ӣзҷә вӯҗNEWвӯҗ
в”Ӯ   в”ңв”Җв”Җ Claude Codeе°Ӯз”Ё
в”Ӯ   в”Ӯ   в”ңв”Җв”Җ Hooksйқ©е‘Ҫ2025
в”Ӯ   в”Ӯ   в””в”Җв”Җ Hooksе®ҹиЈ…з·Ё
в”Ӯ   в””в”Җв”Җ гғһгғ«гғҒгӮЁгғјгӮёгӮ§гғігғҲй–Ӣзҷә
в”Ӯ       в”ңв”Җв”Җ Claude 4Г—GitHub Copilotе®ҹиЈ…
в”Ӯ       в”ңв”Җв”Җ Sonnet 4е®ҹиЈ…гӮ¬гӮӨгғү  
в”Ӯ       в””в”Җв”Җ гғҸгғігӮәгӮӘгғіе®ҹиЈ…гӮ¬гӮӨгғү
в”ңв”Җв”Җ AIй–ӢзҷәеҹәзӨҺ
в””в”Җв”Җ е®ҹи·өгӮ¬гӮӨгғү

рҹ“ҳ MkDocsгғ»гӮөгӮӨгғҲж§ӢзҜү
в”ңв”Җв”Җ еҹәжң¬иЁӯе®ҡгғ»йҒӢз”Ё
в”ңв”Җв”Җ й«ҳеәҰгҒӘж©ҹиғҪиЁӯе®ҡ
в”ңв”Җв”Җ GitHub ActionsзөұеҗҲ
в””в”Җв”Җ гғ—гғ©гӮ°гӮӨгғіжҙ»з”Ё

рҹӣ пёҸ гғ„гғјгғ«гғ»й–ӢзҷәеҠ№зҺҮеҢ–
в”ңв”Җв”Җ гғ—гғӯгӮ°гғ©гғҹгғігӮ°
в”ңв”Җв”Җ Gitгғ»GitHubжҙ»з”Ё
в””в”Җв”Җ й–Ӣзҷәз’°еўғж§ӢзҜү
    ## рҹҺҜ зү№еҫҙгғ»жіЁзӣ®гӮігғігғҶгғігғ„
    ### вӯҗ жңҖж–°иҝҪеҠ гӮігғігғҶгғігғ„ - **AIгӮЁгғјгӮёгӮ§гғігғҲй–ӢзҷәгӮ»гӮҜгӮ·гғ§гғі**: 2025е№ҙжңҖж–°гҒ®AIй–ӢзҷәжүӢжі• - **Claude Code Hooks**: йқ©е‘Ҫзҡ„гҒӘиҮӘеӢ•еҢ–ж©ҹиғҪгҒ®е®Ңе…ЁгӮ¬гӮӨгғү - **гғһгғ«гғҒгӮЁгғјгӮёгӮ§гғігғҲ**: Claude Г— GitHub CopilotеҚ”иӘҝй–Ӣзҷә ### рҹ“Ҡ гӮігғігғҶгғігғ„зөұиЁҲ - **з·ҸиЁҳдәӢж•°**: 500+ иЁҳдәӢ - **гғЎгӮӨгғігӮ«гғҶгӮҙгғӘ**: 6еҲҶйҮҺ - **иЁҖиӘһеҜҫеҝң**: ж—Ҙжң¬иӘһгғ»иӢұиӘһпјҲдёҖйғЁпјү - **жӣҙж–°й »еәҰ**: йҖұж¬Ўжӣҙж–° ### рҹ”Ҙ дәәж°—гғҲгғғгғ—5 1. Claude Code Dockerе®Ңе…ЁгӮ¬гӮӨгғү 2. Amazon Kiroе®Ңе…ЁгӮ¬гӮӨгғү 3. Claude CodeиҮӘеӢ•е®ҹиЎҢгӮ¬гӮӨгғү 4. GitHub CopilotгӮ«гӮ№гӮҝгғ гӮӨгғігӮ№гғҲгғ©гӮҜгӮ·гғ§гғі 5. Linux гғ—гғӯгӮӯгӮ·иЁӯе®ҡе®Ңе…ЁгӮ¬гӮӨгғү
    ## рҹ§ӯ гғҠгғ“гӮІгғјгӮ·гғ§гғіиЁӯиЁҲ ### гғҲгғғгғ—гғ¬гғҷгғ«ж§ӢйҖ  
    рҹ“ҡ е®Ңе…ЁгӮ¬гӮӨгғүйӣҶ вҶ’ йҮҚиҰҒгӮ¬гӮӨгғүгҒ®гҒҫгҒЁгӮҒгғҡгғјгӮё
рҹ”§ Obsidianе®Ңе…ЁгӮ¬гӮӨгғү вҶ’ Obsidianе°Ӯз”ЁгӮ¬гӮӨгғү
рҹ“қ жңҖж–°жӣҙж–° вҶ’ ж–°зқҖгғ»жӣҙж–°иЁҳдәӢдёҖиҰ§
рҹ“қ гғ–гғӯгӮ° вҶ’ гғ–гғӯгӮ°еҪўејҸгҒ®иЁҳдәӢ
    ### рҹ“° гғ–гғӯгӮ°гҒ®дҪҚзҪ®гҒҘгҒ‘пјҲSEOеҗ«гӮҖпјү - гғ–гғӯгӮ°гҒҜгҖҢ**AIжҙ»з”ЁгҒ®е®ҹи·өгғЎгғў**гҖҚгҒ«еҠ гҒҲгҒҰгҖҢ**гғқгӮЁгғ пјҲжҖқиҖғж•ҙзҗҶпјү**гҖҚгҖҢ**зө„з№”и«–пјҲй–Ӣзҷәж–ҮеҢ–гғ»ж„ҸжҖқжұәе®ҡпјү**гҖҚгӮ’жӣёгҒҸгӮігғ©гғ жһ  - гҒҹгҒ гҒ—гӮ«гғҶгӮҙгғӘпјҲnavпјүгӮ’еў—ж®–гҒ•гҒӣгҒӘгҒ„гҒҹгӮҒгҖҒ**еҲҶйЎһгҒҜгӮҝгӮ°дёӯеҝғ**гҒ§йҒӢз”ЁгҒҷгӮӢ - жӨңзҙўжөҒе…ҘгҒҜеҹәжң¬гҖҢиЁҳдәӢеҚҳдҪ“гҖҚгҒ§еҸ–гӮҠгҒ«иЎҢгҒҚгҖҒдёҖиҰ§гғҡгғјгӮёгҒҜеӣһйҒҠе°Һз·ҡгҒЁгҒ—гҒҰдҪҝгҒҶ **гӮҝгӮ°йҒӢз”Ёгғ«гғјгғ«пјҲгҒҠгҒҷгҒҷгӮҒпјү** - гғ–гғӯгӮ°иЁҳдәӢгҒ«гҒҜгҖҒжңҖдҪҺ1гҒӨгҖҢжҹұгӮҝгӮ°гҖҚгӮ’д»ҳгҒ‘гӮӢпјҲдҫӢ: `ai-practice` / `organization` / `poem`пјү - д»»ж„ҸгӮҝгӮ°гҒҜжңҖеӨ§3гҒӨгҒҫгҒ§пјҲгӮҝгӮ°д№ұз«ӢгӮ’йҳІгҒҗпјү ### гӮ«гғҶгӮҙгғӘеҲҘгӮўгӮҜгӮ»гӮ№ - **еӯҰзҝ’зӣ®зҡ„**: рҹ“ҡ еӯҰзҝ’гғ»жғ…е ұ вҶ’ еҹәзӨҺзҹҘиӯҳзҝ’еҫ— - **е®ҹи·өдҪңжҘӯ**: рҹ–ҘпёҸ гӮӨгғігғ•гғ©гғ»йҒӢз”Ё вҶ’ жүӢй Ҷжӣёгғ»гӮігғһгғігғүйӣҶ - **AIй–Ӣзҷә**: рҹӨ– AIй–Ӣзҷәгғ»иҮӘеӢ•еҢ– вҶ’ жңҖж–°жҠҖиЎ“гғ»е®ҹиЈ…гӮ¬гӮӨгғү - **гӮөгӮӨгғҲж§ӢзҜү**: рҹ“ҳ MkDocs вҶ’ гғүгӮӯгғҘгғЎгғігғҲгӮөгӮӨгғҲдҪңжҲҗ - **еҠ№зҺҮеҢ–**: рҹӣ пёҸ гғ„гғјгғ« вҶ’ й–Ӣзҷәз’°еўғгғ»гғҜгғјгӮҜгғ•гғӯгғјж”№е–„
