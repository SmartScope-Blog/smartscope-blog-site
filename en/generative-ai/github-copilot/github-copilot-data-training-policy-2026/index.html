    GitHub Copilot Data Training Policy Change (April 24, 2026) -- How to Opt Out and What's Affected¶

    GitHub Copilot Complete Guide

    For / Key Points

    For: Developers using GitHub Copilot Free / Pro / Pro+

    Key Points:

    • Starting April 24, 2026, Copilot interactions will be used for AI model training by default
    • Business / Enterprise / Student plans are NOT affected
    • You can opt out right now at github.com/settings/copilot
    • Effective April 24 30-day notice with opt-out mechanism
    • Private repos included Code snippets shared during Copilot use are collected
    • 1-minute setup Settings → Copilot → Privacy to disable
    • Tool comparison How Cursor and Claude Code differ

    What's Changing¶

    On March 25, 2026, GitHub announced updates to its Privacy Statement and Terms of Service1. Starting April 24, interaction data from Copilot Free / Pro / Pro+ users will be used for AI model training.

    This is an opt-out (not opt-in) change. If you do nothing, data collection is automatically enabled.

    Affected and Unaffected Plans¶

    PlanAffectedNotes
    FreeYesEnabled by default
    Pro ($10/mo)YesEnabled by default
    Pro+ ($39/mo)YesEnabled by default
    Business ($19/user/mo)NoExcluded by contract
    Enterprise ($39/user/mo)NoExcluded by contract
    Students & Teachers (free Pro)NoExplicitly excluded

    Individual plan users (Free/Pro/Pro+) are the target. Organization-contracted Business/Enterprise remain excluded as before.

    What Data Is Collected¶

    The following "interaction data" becomes subject to collection2:

    • Inputs sent to Copilot (including code snippets)
    • Outputs generated by Copilot (accepted/modified code)
    • Code context around cursor position
    • File names, repository structure, navigation patterns
    • Interactions with Copilot features (chat, inline suggestions)
    • User feedback (ratings)

    Private repositories are included

    Repository content is not used for training "at rest." However, code snippets from private repositories that you actively interact with during Copilot use may be collected and used.

    There is no per-repository control. Opt-out applies to your entire account.

    How to Opt Out (1 Minute)¶

    Settings page: github.com/settings/copilot

    Steps¶

    1. Log in to GitHub
    2. Navigate to github.com/settings/copilot
    3. Scroll to the "Privacy" section
    4. Toggle the training-related option ("Allow GitHub to use my data for AI model training" or similar wording) to off

    If you previously opted out of "GitHub Copilot interaction data for product improvement," that setting carries over. Note that the exact setting name may change with UI updates -- look for the toggle in the "Privacy" section.

    Verification

    After changing the setting, reload the page to confirm the toggle remains off. This is a personal setting independent of your organization's Copilot admin configuration.

    Why This Is Controversial¶

    GitHub Community Discussion #188488 received 232 downvotes, and out of 39 community posts, only one GitHub VP supported the plan3.

    Three main concerns have emerged from the community:

    • Opt-out design criticism: Enabled without explicit user consent. In the EU, GDPR typically requires opt-in, raising legal questions
    • Discoverability of settings: The opt-out toggle has been criticized as a "dark pattern" due to its non-obvious placement
    • No per-repository control: No way to exclude specific private repositories while keeping others included

    GitHub CPO Mario Rodriguez stated that "by participating, you'll help our models better understand development workflows, deliver more accurate and secure code pattern suggestions"1.

    Comparison with Other AI Coding Tools¶

    ToolData Training Policy
    GitHub Copilot (Free/Pro/Pro+)Opt-in by default. Opt-out available
    CursorPrivacy Mode (default OFF) enables zero data retention
    Claude Code (Anthropic)API inputs are not used for model training

    Cursor offers an explicit privacy mode toggle, and Claude Code (Anthropic API) does not use input data for training at all. GitHub cites similar practices from JetBrains, Microsoft, and others to justify its approach1.

    Summary¶

    There's about one month until the April 24 effective date. You can check your opt-out settings right now.

    Take action early if any of the following apply:

    • Using Copilot with a personal account for work code: Risk of proprietary code entering training data
    • Working in private repositories (non-OSS): Code snippets are subject to collection
    • Users in GDPR jurisdictions: You can object to processing based on legitimate interest (contact GitHub's privacy team)

    Migrating to a Business/Enterprise plan is another option, but checking the settings page for opt-out is the quickest step.
