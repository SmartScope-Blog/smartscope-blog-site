    Claude Code 完全ガイド

    Claude Code Web完全ガイド - ブラウザだけで始める新しいクラウド開発環境¶

    Claude Code Webとは¶

    Claude Code Webは、Anthropicが2025年10月に公開したクラウド実行型の開発環境です。Claude Artifactsを基盤として、ブラウザやiOSアプリからコード生成・テスト・レビュー・PR作成まで自動化できます。ローカルPCへのインストールは不要で、GitHub連携とタスク指示だけで開発フローを委任できるのが特徴です1.

    この記事のポイント¶

    特長サマリ

    • クラウド実行: すべてのタスクがAnthropic管理のサンドボックスで走り、依存パッケージの取得やテスト実行まで自動化。ローカル環境のセットアップが不要になります。
    • 並列タスク管理: 複数リポジトリをまたいで同時にタスクを走らせ、個別のPull Requestを生成。バックログ処理やメンテナンスをまとめて進めやすくなります。
    • モバイル対応: iOSアプリでも同じWebインターフェースが利用でき、現場訪問やリモートワーク中でもPR作成・レビューが可能です。
    • Artifactsとの連携: Claude Artifactsで生成したUI・ドキュメントを即座に呼び出し、仕様書や画面ファイルをブラウザだけで用意できます。
    • 安全な委任: gVisor相当のサンドボックスでネットワーク・ファイルアクセスを制御し、認可したリポジトリ以外の情報にはアクセスできません。

    代表的なユースケース¶

    • バグ修正・保守タスクの自動化: GitHub IssuesのURLや再現条件を指定しておけば、修正コードとテストを生成してPRを仕上げます。
    • リポジトリ調査とドキュメント整備: 「このサービスの主要エンドポイントをまとめて」「変更影響範囲を説明して」といった調査系タスクを任せられます。
    • リファクタリングやコードモダナイズ: 複数リポジトリを同時に処理し、用途ごとにPRを分割。モノレポ移行やライブラリアップデートの効率化に有効です。
    • モバイルからの緊急対応: 外出先でもiOSアプリからタスクを投入し、CIに通ったPRをそのままマージできます。

    既存ツールとの違い¶

    ローカル版は手元の環境で実行するためカスタムツール呼び出しや低レイテンシ操作が得意。Web版はセットアップ不要で、マシンスペックに依存しません。

    ブラウザから複数タスクを同時に走らせ、PRを生成するワークフローが標準化されています。Artifactsと組み合わせた資料生成や、モバイル操作にも対応します。

    Sandboxで権限を細かく制御できる点が強み。独自のlaws設定やAgent Skillsを使えば、業務専用の振る舞いを追加できます。

    セキュリティと制御¶

    AnthropicはClaude Code Webで以下の保護策を提供しています1: - 各タスクを隔離されたサンドボックスで実行し、許可されたドメインとリポジトリにのみアクセス - GitHubとのやり取りはプロキシ越しに行われ、アクセストークンはClaudeに露出しない - カスタムネットワーク許可リストを設定し、npmやPyPIなど必要なパッケージリポジトリだけを開放可能 - 監査ログやタスク履歴をダウンロードし、社内のセキュリティ体制に合わせた管理が可能

    料金と提供状況¶

    • 提供プラン: 研究プレビュー段階ではClaude ProおよびMaxプランで利用可能。Team/Enterpriseプランは段階的に対応が発表される予定です。
    • 課金・レート制限: Web版の実行は他のClaude Code機能と同じレート制限にカウントされます。大量の並列タスクを走らせる場合はMaxプランが推奨です。
    • 対応デバイス: 最新のChrome/Safari/Edgeなど主要ブラウザ、およびiOSアプリ。Android対応は順次案内予定とされています。

    使い方：セットアップからPR作成まで¶

    ステップバイステップ
    1. claude.com/code にアクセスし、Claudeアカウント（Pro/Max）でログイン
    2. 「Connect a repository」からGitHub連携を許可し、対象リポジトリとブランチを選択
    3. タスクカードで依頼内容（例: 「tests/ ディレクトリのFlakyテスト修正」）や制約、テストコマンドを指定
    4. 実行中はブラウザ上で進捗ログを確認し、指示の修正や追加をリアルタイムで送信可能
    5. 完了すると変更内容をまとめたPRが作成され、テスト結果や差分サマリも自動で添付されます
    6. iOSアプリからも同じ手順でタスクを投入・確認でき、移動中の軽微な対応に便利です

    よくある質問¶

    Claude Code Webは無料で試せますか？

    Pro/Maxプランの契約が必要です。研究プレビュー中でも追加料金は発生せず、通常の利用枠に含まれます。

    ローカルCI/CDとの連携は？

    GitHub Actionsなど既存のCIが走る前提でPRを作成します。プルリク作成後にCIが自動で検証し、失敗した場合はClaudeに再修正を依頼できます。

    社内リポジトリでも安全に使えますか？

    許可したリポジトリとドメインに限定してアクセスが行われます。Sandboxログの監査とIP制限を組み合わせることで、社内ポリシーに合わせた運用が可能です。

    CodexやCopilotと併用できますか？

    問題ありません。Claude Code Webで生成したPRをCopilotでレビューしたり、Codexで補助スクリプトを作成するといった併用例も想定されています。
