      📖 Claude Codeとは¶

      Claude Codeは、Anthropic社が開発した最先端のAIコーディング支援プラットフォームです。2025年8月の大幅アップデートにより、単なるチャットツールから包括的な開発支援環境へと進化しました。

      主な特徴¶

      • 世界最高レベルの性能: Claude Opus 4.6はTerminal-Bench 2.0・Humanity's Last Examで業界首位、Claude Sonnet 4.5はSWE-bench 77.2%・OSWorld 61.4%を達成（2026年2月時点）
      • 1,000,000トークンのコンテキスト: 大規模プロジェクト全体を理解（Claude Sonnet 4以降）
      • マルチモーダル対応: コード、画像、ドキュメントを統合的に処理
      • 日本語完全対応: 自然な日本語での対話とコード生成
      • Haiku 4.5: 高速処理向けの軽量モデルも利用可能

      v2.0の主要新機能（2025年後半〜2026年）¶

      機能概要
      Compactionコンテキスト圧縮により、長時間セッションでも重要な情報を保持。トークン消費を抑えつつ品質を維持
      Agent Teams複数のサブエージェントが並列で作業し、複雑なタスクを分割して高速処理
      VS Code深層統合エディタ内でのインライン差分表示、ターミナル統合、ファイル操作のシームレスな連携
      Checkpoint / Restore作業状態の保存・復元により、安全な試行錯誤が可能
      14種類のHookイベントPreToolUse、PostToolUse、Notificationなど、開発ワークフローの各段階で自動化を実現
