      🎛️ インストラクション体系¶

      Copilotにプロジェクトの知識を伝える仕組みは4つの階層＋2つの拡張で構成されています。

      仕組み適用タイミング典型的な用途
      copilot-instructions.md常時（全リクエスト）コーディング規約、技術スタック
      .instructions.md + applyToファイルパス一致時（自動）フロント/バックエンド別ルール
      Agent Skillsエージェントが必要と判断したときTerraformレビュー手順、リリース手順
      AGENTS.mdエージェント起動時プロジェクト全体の方針（マルチツール共通）

      • カスタムインストラクション完全ガイド

        .github/copilot-instructions.md と .instructions.md の設定方法、インストラクション体系の全体像、ベストプラクティス

      • applyToパターンガイド

        ファイルパスに基づく条件分岐型インストラクション。excludeAgent によるエージェント別制御

      • Agent Skillsガイド

        SKILL.md フォーマット、Progressive Disclosure、オンデマンド起動の手順知識パッケージ

      • AGENTS.md統一管理ガイド

        GitHub Copilot・Claude Code・Cursor等で共通利用できるプロジェクト命令とシンボリックリンク戦略

      運用の考え方は Copilot方法論ガイド（Skills / AGENTS.md の導入判断基準、Claude Codeとの役割分担）も参照してください。

      🤖 エージェント機能¶

      Copilotはコード補完を超え、自律的にタスクを実行するエージェントとして機能します。

      • Coding Agent完全整理

        Coding AgentとAgent Modeの違い、向くタスク、PR前セルフレビュー、コスト設計を実務目線で整理。

      • マルチエージェント協働ガイド

        Agent Mode・Coding Agent・サードパーティ統合の全体像、料金体系、最新モデル情報

      • Hooks完全ガイド

        preToolUseによるツール実行制御、監査ログ、セキュリティ強制の実装パターン

      • プレミアムリクエスト節約術

        月間枠を効率的に使い切る節約術5選、消費が加速する8つのアンチパターン

      過去のアップデート記録: Agent Mode & Claude Code（2025年8月版）

      Agent Skillsはエージェントが自律実行する際の手順知識としても機能します。詳細はインストラクション体系のAgent Skillsカードを参照してください。

      プレミアムリクエストの消費はモデルによって異なります。プラン別の回数と含まれるモデルは以下の通りです。

      料金プラン（2026年2月時点） — 無料（50回/月）〜Enterprise（1,000回/user/月）。Agent Modeは全プランで利用可能
      プラン月額プレミアムリクエスト含まれるモデル（リクエスト非消費）主な対象
      Free$050回GPT-4o, GPT-4.1個人・学生
      Pro$10300回GPT-4o, GPT-4.1, GPT-5 mini個人開発者
      Pro+$391,500回GPT-4o, GPT-4.1, GPT-5 miniヘビーユーザー
      Business$19/user300回/userGPT-4o, GPT-4.1, GPT-5 miniチーム
      Enterprise$39/user1,000回/userGPT-4o, GPT-4.1, GPT-5 mini大規模組織
      • Agent Modeは全プランで利用可能
      • プレミアムリクエストは高性能モデル（Claude Opus 4.6, GPT-5.1, Gemini 3 Pro等）使用時に消費。基本的なコード補完は無制限
      • 追加購入: $0.04/リクエスト
