      Google Antigravity徹底レビュー - AI駆動型IDEの実力と最新評価¶

      この記事の対象者

      • AI駆動型の開発ツールに関心がある開発者
      • Cursor・Copilot等の競合ツールと比較検討したい方
      • Antigravityの導入...

      Google Antigravityとは何か¶

      2025年11月18日、GoogleはGemini 3 Proの発表と同時に、新しいAI駆動型IDE「Antigravity」を公開しました。従来のコード補完ツールとは一線を画す「エージェントファースト」のアプローチで、AIが自律的に計画立案、コード生成、テスト、デバッグ、さらにはブラウザ操作まで実行します。

      AntigravityはAI IDE「Windsurf」の技術をベースとしており、GoogleがWindsurf開発元から技術ライセンスを取得し創業チームを雇用した経緯がある。VS Code互換のデスクトップアプリケーションとしてWindows・macOS・Linuxに対応し、既存のVS Code拡張機能やキーバインド設定との互換性が高く、移行時の学習コストを大幅に削減できます。現在は無料のパブリックプレビュー版として提供されており、Gemini 3 Pro / Deep Thinkを中心に、Claude Opus 4.5やGPT-OSS-120Bなど複数のAIモデルをサポートしています。

      開発者の生産性を劇的に向上させることを目指したこのツールは、リリース直後からX（旧Twitter）を中心とした開発者コミュニティで大きな反響を呼んでいます。本記事では、公式情報とリアルタイムのユーザーフィードバックを基に、Antigravityの実力を徹底検証します。

      Antigravityの主要機能¶

      Editor View & Manager View - 2つのワークフロー¶

      Antigravityは用途に応じて2つのインターフェースを提供します。

      Editor View（エディタビュー） 従来のIDEライクな画面で、単一エージェントがサイドパネルで動作。CursorやGitHub Copilotに近い使用感で、同期的なコーディング作業に適しています。

      Manager View（マネージャービュー） 複数のエージェントを同時に管理できる「ミッションコントロール」。異なるワークスペースで複数のタスクを並行実行させ、非同期でプロジェクトを進める高度な使い方が可能です。

      開発者からは「干渉なしで効率的に作業できる」「別ウィンドウでエージェント管理できるDXが最高」との声が上がっています。

      Browser Subagent - 自律的なブラウザ操作¶

      Antigravityの最大の特徴の1つが、AIがブラウザを自律操作できる機能です。UIテストやフロントエンドの動作検証を、エージェントが自動で実行します。

      従来はPlaywrightなどのサードパーティツールが必要でしたが、Antigravityではこれが統合されています。開発者コミュニティでは「Playwright MCPよりagentic」「game changer」との評価を得ています。

      Browser Subagentの活用例

      • フォーム送信の動作確認
      • レスポンシブデザインのチェック
      • JavaScriptエラーの自動検出

      Artifacts機能 - 透明性の高い実装記録¶

      Antigravityは実装プロセスを可視化する「Artifacts」機能を備えています。エージェントが生成した計画書、スクリーンショット、ブラウザ操作の録画を自動保存し、後からレビューできます。

      「Walkthrough機能がinsane、録画で実装フローを把握できる」との声があり、チーム開発やコードレビューでの透明性確保に貢献します。

      マルチモデル対応 - 柔軟なAI選択¶

      Antigravityは複数のAIモデルをサポートしています。

      モデル用途強み
      Gemini 3 Proメインコーディング高速（128トークン/秒）、推論力
      Gemini 3 Flash軽量タスク低レイテンシ、コスト効率
      Gemini 3 Deep Think複雑な推論深い思考、難問解決
      Claude Opus 4.5（Thinking）複雑なタスク安定性、精度
      GPT-OSS-120B汎用タスクオープンソース、幅広い対応力
      Nano Banana (Gemini 2.5 Flash Image)画像生成・編集UI素材作成、キャラクター一貫性

      開発者は用途に応じてモデルを切り替えられます。X上では「Gemini 3 Proが3000行を3分で生成、insanely fast」との報告がある一方、「複雑タスクではClaude Opus 4.5の方が安定」との意見も見られます。

      競合ツールとの比較¶

      Antigravityは既存のAI駆動型開発ツールとどう違うのでしょうか。主要な競合ツールと比較します。

      Cursorとの比較¶

      CursorはAIコード補完とチャットベースのエディタとして人気があります。Background AgentsやHooksが追加され、SOC 2認証も取得済み。$10B評価額に到達し、市場のリーダー的存在です。

      Antigravityの優位点 - ブラウザ統合とArtifactsによる透明性 - マルチモデル対応で柔軟性が高い - 無料プレビューで制限が緩い

      Cursorの優位点 - UIの洗練度とエディタとしての完成度 - SOC 2認証済みでエンタープライズ導入に強い - Background Agentsによる非同期タスク実行

      開発者の声：「CursorからAntigravityへのスイッチを検討、ブラウザテストがwild」「UIはCursorの方がクリーン」

      GitHub Copilotとの比較¶

      GitHub Copilotはリアルタイム補完に強みがあります。

      Antigravityの優位点 - エージェントの自律性（計画→実装→テスト） - Artifacts機能で実装プロセスが可視化

      Copilotの優位点 - GitHub統合の深さ - 安定性と実績

      開発者の声：「Copilotのローカル版みたいだが、git worktree並行開発が欲しい」

      Windsurfとの比較¶

      WindsurfはVS Code系のGUI IDEで、現在はCognition（Devin）に買収されている。Gartnerリーダー認定を受けており、エンタープライズ導入の実績がある。

      Antigravityの優位点 - Artifacts（証跡）による作業の構造化と可視化 - Browser Subagentによるブラウザ検証の統合 - Manager Viewでの複数エージェント管理

      Windsurfの優位点 - 市場での成熟度とエンタープライズ実績 - Cognition（Devin）統合による自律性の高さ

      Claude Codeとの比較¶

      Claude CodeはCLI型のエージェントツールで、ターミナルから直接操作する。Agent Teams（experimental）やHooks、subagent機能が追加され、大幅に進化している。

      Antigravityの優位点 - ブラウザ操作と録画機能 - GUIでの視覚的な管理 - 設定画面での権限管理（ガバナンスUI）

      Claude Codeの優位点 - CLIのシンプルさとCI/CD統合の柔軟性 - Hooksによる決定論的な制御 - レート制限が緩い

      開発者の声：「ClaudeのCLIがクリーンだが、AntigravityのUIがoverwhelming」「レート制限でClaude安定」

      ベンチマークデータについて

      WebDev Arenaで1487スコアを記録するなど、部分的なベンチマーク結果が出始めている。X上では「Gemini 3 ProのコーディングでClaude Opus 4.5超え」との声がある一方、「複雑タスクではClaude Opus 4.5やGPT-5の方が安定」との指摘もある。

      実際の評価とコミュニティの声¶

      リリース直後（2025年11月18〜19日）のX投稿を分析すると、約70%がポジティブ、約30%がネガティブな評価でした。

      ポジティブな評価¶

      革新性と速度 「flawless and intuitive、Gemini 3 Proでworkflow seamless。新ベンチマークになる」 「Twitterクローン+finance trackerをminutesで無料構築」

      ブラウザ統合 「100K行リポをone-shot移行、ブラウザ統合game changer」 「Agentがbrowser input捕獲、impressed so much」

      ネガティブな評価¶

      安定性の課題 「複雑タスクでstruggle、複数イテレーション必要。エラー頻発、production-readyじゃない」 「昨日中途プロジェクト引き継ぎでSonnet 4.5以下。エラーでquota limit」

      UI/UXの問題 「UI cluttered、Claude CodeのCLIに慣れるとoverwhelming」 「限額すぐ到達」

      エコシステムの懸念 「Googleチームの非協調（Jules/Antigravity重複）、エコシステム乱れ」

      総合評価¶

      ポテンシャルは非常に高い（★⅘相当）ものの、安定性向上が課題です。興味深いのは、「AIバブル懐疑派」からも「実際に使ってimpressed」と評価が転じている点です。

      ⚠️ セキュリティと安全性に関する注意¶

      Antigravityのエージェントファースト設計には、安全性に関する重要な課題も報告されている。

      報告されたインシデント

      • 2025年11月、ユーザーがキャッシュクリアを依頼した際、エージェントがDドライブ全体を削除した事例
      • セキュリティ研究者によるRCE（リモートコード実行）を含む複数の脆弱性の報告

      推奨される対策

      • 重要なデータのバックアップを事前に取得
      • エージェントの権限設定（ガバナンスUI）で実行範囲を制限
      • 本番環境への直接接続を避ける
      • 詳細はAntigravityの強みと向き不向きを参照

      2026年1月以降のコミュニティ動向¶

      リリース直後の熱狂の後、2026年1月にはパフォーマンス品質の低下がコミュニティで大きな議論になった。Redditでは「Antigravityが$20のペーパーウェイトになった」というスレッドがバイラル化。コンテキストウィンドウの縮小、ハルシネーションの増加、バックエンドモデルの差し替え疑惑などが指摘されている。

      AI IDEの進化速度は極めて速く、評価は流動的である点に留意が必要。

      料金体系と今後の展望¶

      現在の料金（2026年2月時点）¶

      • 無料枠: パブリックプレビューとして無料利用可能（レート制限あり）
      • Google AI Pro（$20/月）: 2TB Storage + Gemini Advanced含む、優先アクセスと緩和されたレート制限
      • レート制限: 無料枠ではリリース当初より厳しくなっているとの報告あり

      公式ドキュメント（https://antigravity.google/pricing）では「無料プレビュー製品、制限あり」と明記。正式版では価格体系の改定が予想され、現時点のコスト面だけで選定すべきではない。

      導入手順¶

      Antigravityの導入は以下の3ステップで完了します。

      1. 公式サイトからダウンロードhttps://antigravity.google/download からOS別のインストーラーを取得

      2. Googleアカウントでサインイン 既存のGoogleアカウントでログイン（ウェイトリストなし）

      3. エージェントモードを有効化 設定画面から「Enable Agent Mode」をONにして使用開始

      地域制限について

      個人Gmailアカウントで広く利用可能になっている。地域制限は緩和されており、多くの地域でVPNなしでアクセスできるとの報告がある。

      今後の予定¶

      • Gemini 3 Deep Thinkが利用可能になり、複雑な推論タスクに対応
      • Agent Skills機能が追加され、再利用可能な「プレイブック」として汎用モデルを特定タスクの専門家に変換可能
      • git worktree機能の追加を求める声が多い
      • エラー修正とUI改善により「業界標準になる」との期待

      X上では「エラーフィックスで業界標準に」との期待が高まっています。日本語コミュニティの投稿は少なく、グローバル展開が待たれます。

      まとめ¶

      Google Antigravityは、AI駆動型IDE市場に新しい風を吹き込む可能性を秘めたツールです。

      向いている人 新しいAIツールを積極的に試したい開発者 ブラウザ統合やマルチエージェント管理に魅力を感じる方 無料でGemini 3 Proを活用したい方

      まだ様子見が良い人 安定性を最優先する本番環境での使用 UIのシンプルさを重視する方 レート制限に余裕が欲しい方

      リリース直後で荒削りな部分はあるものの、Googleのエコシステムと高速なGemini 3 Proを活用できる点は大きな強みです。ただし、セキュリティ上の課題も報告されており、特に本番環境での使用には慎重な判断が必要です。今後のアップデートに期待しつつ、まずは無料プレビューで実際に試してみることを推奨します。
