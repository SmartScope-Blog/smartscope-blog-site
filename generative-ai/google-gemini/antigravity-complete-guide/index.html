      Google Antigravity完全ガイド¶

      この記事のポイント¶

      • エージェント監督型のUI設計

        エディタ補助ではなくAgent Manager（Mission Control）が製品の中核を担うアーキテクチャ

      • Browser Subagent

        分離Chromeプロファイルで動作するブラウザ自動操作エージェント

      • ガバナンスUI

        Terminal Sandboxing、URL Allowlist、Strict Modeなど、コードを書かずに設定画面から権限制御

      • Agent Skills

        SKILL.mdベースの能力パックをProgressive Disclosureで自動選択・発動

      📖 Antigravityとは¶

      Google Antigravityは、Geminiモデルを搭載したAI駆動型の統合開発環境だ。従来のAI IDEがエディタにAIチャットを補助として追加する構造なのに対し、AntigravityはAgent Managerをプロダクトの中核に据え、管理→証跡→検証→レビューを1つの運用単位として統合している。

      主な特徴¶

      • Manager View: 複数ワークスペース×複数エージェントを同時監督するダッシュボード
      • Browser Subagent: Gemini 2.5 Pro UI Checkpointによる自律的なブラウザ操作
      • Artifacts: タスクリスト・実装計画書・スクリーンショット・録画を自動構造化
      • ガバナンスUI: ターミナル/ブラウザ/ワークスペースの権限をGUIで設定
      • Agent Skills: SKILL.mdベースの拡張能力パック
      • VS Code互換: 拡張機能・キーバインドの移行が可能
      • マルチモデル対応: Gemini 3 Pro、Deep Think、Claude Opus 4.5等を用途別に選択

      2026年3月時点の主要仕様¶

      項目内容
      ベースVS Code互換エディタ + Agent Manager
      ブラウザ操作分離Chromeプロファイル、DOM capture、動画記録
      セキュリティStrict Mode（ワークスペース分離＋ネットワーク遮断）
      プラン無料プレビュー / Google AI Pro ($20/月)
      対象個人Gmailアカウント（エンタープライズ展開は進行中）
