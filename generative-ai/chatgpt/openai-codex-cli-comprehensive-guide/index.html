      🚀 Codex CLIとは？¶

      Codex CLIは、OpenAIが開発した公式のコマンドライン型AIコーディングツール。ターミナル内で動作し、自然言語の指示でプログラミングタスクを自動化するエージェントだ。

      特徴詳細
      認証ChatGPT Plus/Pro/Team認証で追加料金なし、またはAPIキー課金
      実行環境ローカル実行。必要な差分・抜粋のみAPI送信
      安全性3段階の実行モード（Read Only / Auto Edit / Full Auto）
      隔離macOS Seatbelt、Linux Landlock+seccomp によるOSレベルサンドボックス
      ライセンスApache-2.0（オープンソース）

      セットアップ手順は 概要とクイックスタート を参照。

      🆕 2026年2月 最新情報

      • GPT-5.3-Codex: 最新のエージェント型コーディングモデル。GPT-5.2比25%高速化
      • GPT-5.3-Codex-Spark（研究プレビュー）: リアルタイムコーディング向け軽量モデル。1000+ tokens/sec
      • Codex CLI v0.101.0+: サンドボックス安定性向上、プロファイル永続化改善
