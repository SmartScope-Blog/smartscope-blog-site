    Codex CLI 完全ガイド

    ChatGPT Atlas完全ガイド - ChatGPT搭載AIブラウザで広がる新しいWeb体験¶

    Atlas Browserとは¶

    ChatGPT Atlasは、ChatGPTをブラウザにネイティブ統合したmacOS先行提供のAIブラウザです。初回リリースは2025年10月で、ブラウザ内から直接AIエージェントにタスクを委任し、フォーム入力・情報検索・要約・ショッピング補助などを自動化できます。Windows、iOS、Android版も順次リリース予定と公式が案内しています1.

    インストール手順を探している方へ

    macOSでの導入方法は「ChatGPT Atlasインストールガイド（macOS）」で詳しく解説しています。

    主な機能と特長¶

    AIエージェントによるブラウザ制御¶

    • 任意のタブで会話しながら、サイト内操作（ナビゲーション、ボタン押下、フォーム入力）や情報抽出を自動化
    • プロンプトに業務ルールを設定すると、Atlas側でコンテキストを保持しながら連続タスクを処理
    • Agent modeはChatGPT Plus / Pro / Businessユーザーでプレビュー提供され、Free / Goプランでは利用できません3

    パーソナライズドメモリ¶

    • Atlas内で行った会話・操作を安全に保存し、ユーザーの好みや履歴をもとに次回提案を最適化
    • メモリのON/OFFや削除はダッシュボードから即時操作可能で、用途に応じて切り替えられます
    • 既定では閲覧内容がモデル学習に使用されず、「Include web browsing」を有効化した場合のみデータが学習利用されます4

    マルチモーダル検索とコンテンツ理解¶

    • テキスト／画像／動画を問わず、ページ全体を解析して要約や比較表を生成
    • Atlas上で表示中の資料をスクリーンショットなしで引用・共有し、レポート作成を省力化

    拡張機能・テーマ対応¶

    • Extensionsメニューから拡張機能の有効化・無効化・削除を制御。Chrome Web Store経由の多くの拡張が動作する一方、native messagingなど一部機能が必要な拡張は非対応または不安定な場合があります5
    • テーマやレイアウトをカスタマイズして、ダークモードや集中モードをワンクリックで切り替え

    冒頭で押さえたいポイント

    • AIエージェントがブラウザ操作を自動化し、プロンプトだけでタスクを進行
    • メモリ機能により、次回以降の提案や補助がユーザー仕様に最適化
    • Extensions対応で既存のChromeワークフローを移行しやすいが、互換性は100%ではない点に注意

    セキュリティとプライバシー¶

    保護設計のハイライト

    • サイトごとの権限管理、インコグニトモード、履歴クリアを標準搭載
    • Cookie・トラッキング拒否設定をプロファイル単位で適用
    • ページ可視性トグルとエージェント権限制御により、コード実行・ファイルダウンロード・拡張インストール・保存済みパスワードへのアクセスはエージェントから行えません6
    • 既定で閲覧情報は学習利用されず、共有設定でオプトインした場合のみモデル改善に反映されます

    他ブラウザ／AIツールとの比較¶

    比較表（Atlas vs 他サービス）
    項目ChatGPT AtlasChrome + 拡張AIPerplexity / Arc等
    AIエージェント制御ブラウザに常駐。タブ操作やフォーム入力まで自動化（権限制限あり）拡張機能で補助。操作自動化は限定的サイト要約や検索支援が中心
    メモリ機能パーソナライズ設定をブラウザ全体で共有（既定は学習オプトアウト）拡張ごとに保持。統合が難しい質問履歴ベースの提案
    プライバシー設定ページ可視性トグルとエージェント権限制御を統合管理ブラウザ本体と拡張で分離管理サービス側ポリシー依存
    クロスプラットフォームmacOS→Windows/iOS/Androidに拡張予定全OS対応サービスにより異なる
    企業向け機能SSO/SCIM/監査ログは未対応。MDMで一部ポリシー制御GWS/Azure ADなどのSSOが利用可能監査・データ管理はサービス次第

    使ってみる前に準備すること¶

    • Apple silicon搭載のMacで、macOS 12 Monterey以降が必要です2。公式サイトからインストーラをダウンロードし、ChatGPTアカウントでログイン
    • 個人・法人どちらでも使用できるが、チーム管理を行う場合はOpenAIアカウントに管理者権限が必要
    • 業務利用では、社内のセキュリティポリシーに合わせて履歴保存・メモリ機能の設定を見直す

    Atlas Browserの使い方（クイックスタート）¶

    初回タスクの流れ
    1. Atlasを起動し、ログイン後にホーム画面の「ニュータスク」からAIエージェントの指示パネルを開く
    2. プロンプトに「明日の東京→大阪の最安移動手段を調査し、リンク付きでまとめて」と入力
    3. Atlasが複数タブで予約サイト・比較サイトを横断し、結果をサマリカードで提示
    4. 必要に応じて追加質問や修正指示を送り、完成した結果をPDFまたはメモリに保存

    コミュニティの初期反応¶

    • 公式の発表ポストはリリース動画付きで大きな反響を集め、AIアシスタントがタブを操作するインタラクションが注目されました8.
    • セキュリティ強化やChromeとの乗り換え議論が技術者の間で活発化し、マルチプラットフォーム展開への期待も高まっています。

    よくある質問¶

    Atlasは無料で使えますか？

    ChatGPT Atlas自体は無料プランでも利用可能です。Agent modeなどの高度な機能はPlus / Pro / Businessのユーザー限定でプレビュー提供されています3.

    既存のブックマークやパスワードは移行できますか？

    ChromeやSafariからブックマーク・履歴・パスワードをインポート可能です。Atlas内のパスワード管理機能で暗号化されます。

    企業で導入する際のポイントは？

    現時点ではAtlas専用のSSO強制・SCIM・監査ログエクスポートは提供されていません。MDM経由のポリシー制御など、既存の管理基盤と組み合わせたガバナンス設計が必要です7.

    拡張機能はどこまで使えますか？

    ExtensionsメニューからChrome Web Storeの拡張を追加できます。多くは動作しますが、native messagingなど高度な権限を必要とする拡張は未対応の場合があります5.

    今後のロードマップと活用ヒント¶

    ロードマップを活かす

    • Windows、iOS、Androidアプリがリリースされれば、Atlasをチーム全体の共通AIポータルとして活用可能
    • Agent Skillsや自動化ワークフローと組み合わせて、社内ポータルやナレッジベース更新をAtlasから一元化すると効果的です
    • 競合サービスの動向（Perplexity Browser、Arcなど）もチェックし、ブラウザごとの得意分野を見極めましょう
