      📚 カテゴリー別ガイド¶

      • Claude (Anthropic)

        高精度・長文処理と安全性に強み。 - GitHub Copilot

        コード補完とAIペアプログラミング標準。 - ChatGPT (OpenAI)

        汎用性と拡張プラグインで幅広い応用。 - Amazon Kiro

        エージェントIDE形態で統合開発を効率化。 - Google Gemini

        マルチモーダルと長文コンテキスト対応。 - Grok (xAI)

      🎯 用途別選び方ガイド¶

      • コーディング支援: GitHub Copilot + Claude Code
      • 推奨: Copilot総合, Copilotベストプラクティス, Claude Code完全
      • 使い分け: 日常補完=Copilot / 大規模改変・仕様化=Claude Code
      • 仕様策定・長文要約: Claude（Sonnet/Opus系）
      • 推奨: Claude総合, モデル: Sonnet 4 → 長文/多段推論、Opus → 最精度
      • 研究・ライティング・アイデア拡張: ChatGPT
      • 推奨: ChatGPT総合, プラグイン/エージェント活用で探索力強化
      • 🔥 新着: Codex CLI 自動承認モード完全ガイド - Claude Codeの--dangerously-skip-permissions相当機能を実現
      • 企業内エージェント/IDE統合: Amazon Kiro
      • 推奨: Kiroガイド, 権限/レビュー/自動運用の統制に強み
      • マルチモーダル（画像/音声/動画）: Google Gemini
      • 推奨: Geminiガイド, 入力多様性と長コンテキスト
      • リアルタイム/ソーシャル連携: Grok
      • 推奨: Grokガイド, X連携と最新情報に最適
